# Hollow Knight Style Demo - Godot 游戏

一个基于 Godot 4.x 引擎制作的小型 2D 平台动作游戏 Demo，灵感来源于《空洞骑士》。

## 功能特性

- **WASD** 控制移动，**空格** 跳跃
- **鼠标左键** 攻击（剑击，有冷却时间）
- **敌人 AI**：缓慢巡逻，发现玩家后冲刺追击
- **接触伤害**：敌人碰到玩家会造成伤害
- **生命值系统**：5点生命，受伤后短暂无敌
- **胜利条件**：移动到地图最右侧的绿色光柱区域

## 运行方式

1. 下载并安装 [Godot 4.3+](https://godotengine.org/)
2. 用 Godot 打开本项目文件夹
3. 点击运行（F5）即可

## 项目结构

```
├── project.godot          # 项目配置文件
├── icon.svg               # 项目图标
├── scripts/
│   ├── player.gd          # 玩家控制器
│   ├── enemy.gd           # 敌人AI
│   ├── win_zone.gd        # 胜利区域
│   └── level.gd           # 关卡管理（地图构建）
└── scenes/
    ├── player.tscn        # 玩家场景
    ├── enemy.tscn         # 敌人场景
    └── level.tscn         # 关卡场景
```

## 操作说明

| 按键 | 功能 |
|------|------|
| W / A / S / D | 上下左右移动 |
| 空格 | 跳跃 |
| 鼠标左键 | 攻击 |
| R | 胜利后重新开始 |

## 游戏截图

游戏使用程序生成的像素风格角色：
- 蓝色：玩家角色（带小剑）
- 暗红色带角：敌人（发现玩家时变亮冲刺）

地图从左侧起始区域一直延伸到右侧终点，中间有多个平台和敌人把守。
